# ML Traffic Detection System

## Overview
This project detects and analyzes traffic using machine learning and YOLOv8.

## Features
- Traffic detection from video
- Model training support
- Dataset generation

## Tech Stack
- Python
- OpenCV
- YOLOv8

## How to Run
pip install -r requirements.txt
python main.py

## Note
Model files are not included due to size. Train using train_model.py
